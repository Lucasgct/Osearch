== Starting OpenSearch

The method for starting {opensearch} varies depending on how you installed it. 

[discrete]
[[start-targz]]
=== Archive packages (`.tar.gz`)

If you installed {opensearch} with a `.tar.gz` package, you can start {opensearch} from the 
command line.

[discrete]

[discrete]
[[start-zip]]
=== Archive packages (`.zip`)

If you installed {opensearch} on Windows with a `.zip` package, you can start {opensearch} from 
the command line. If you want {opensearch} to start automatically at boot time without 
any user interaction, <<windows-service,install {opensearch} as a service>>.

=== Docker images

If you installed a Docker image, you can start {opensearch} from the command line. There 
are different methods depending on whether you're using development mode or 
production mode. See <<docker-cli-run>>.
