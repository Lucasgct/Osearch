* If you are running {opensearch} with `systemd`:
+
[source,sh]
--------------------------------------------------
sudo systemctl stop opensearch.service
--------------------------------------------------

* If you are running {opensearch} with SysV `init`:
+
[source,sh]
--------------------------------------------------
sudo -i service opensearch stop
--------------------------------------------------

* If you are running {opensearch} as a daemon:
+
[source,sh]
--------------------------------------------------
kill $(cat pid)
--------------------------------------------------
